Decomatch Pro is a modern, AI-driven interior design marketplace designed to streamline the renovation experience. Built with a robust tech stack including React (Vite), Tailwind CSS, and Supabase, it bridges the gap between homeowners and creative professionals.

Unlike traditional directories, Decomatch Pro leverages Google Gemini AI to provide actionable intelligence. Key features include:

AI Budget Estimator: Instantly calculates detailed renovation costs based on room type, area, and quality preferences.

Smart Comparison: Uses Generative AI to analyze and compare multiple designer profiles, helping users make data-driven hiring decisions.

Intelligent Chatbot: An always-on assistant for style advice and site navigation.

The application features a responsive, glassmorphism-inspired UI with Framer Motion animations, dark mode support, and secure role-based authentication. It represents a complete solution for discovering, verifying, and booking top-tier interior design talent.
